## webサーバ

★☆★☆★☆★☆★☆★☆★☆★☆★☆★☆★☆★☆★☆★☆★☆

***
#### ◎アプリケーションのインストール

```
yum -y install httpd  // webサーバのアプリ

systemclt start httpd // webサーバの起動

systemclt stop httpd // webサーバを停止
```

***
#### 設定の変更

設定ファイルのバックアップ
```
cd /etc/httpd/conf
cp httpd.conf httpd.conf.bkup // copy
```

設定ファイルの操作
```
vi httpd.conf
86g // 86行目に移動
// ServerAdmin を変更
// ServerName を変更
150g // 150行目に移動
// AllowOverride All に変更
GG // 最終行に移動
KeepAlive On を追加（セッション）
```
